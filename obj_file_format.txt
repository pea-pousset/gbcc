File header
Sections

File header:
+======+======+============================+===================================+
| Size | Type | Description                | Value
+======+======+============================+===================================+
| 8    |      | Signature                  | must be "GBOBJECT"                |
+------+------+----------------------------+-----------------------------------+
| 4    | u32  | File format version number | 1                                 | 
+------+------+----------------------------+-----------------------------------+
    
Block header
+======+======+============================+===================================+
| Size | Type | Description                | Value
+======+======+============================+===================================+
| 2    | u16  | Type                       | 0 = sections, 1 = symbols         |
+------+------+----------------------------+-----------------------------------+
| 4    | u32  | Number of entries          |                                   | 
+------+------+----------------------------+-----------------------------------+


Section entry:
+======+======+============================+===================================+
| Size | Type | Description                | Value                             |
+======+======+============================+===================================+
| 4    | u32  | section id                 |                                   |
+------+------+----------------------------+-----------------------------------+
| 2    | u16  | section type               | 0 = .org (fixed address)          |
|      |      |                            |                                   |
+------+------+----------------------------+-----------------------------------+
| 2    | u16  | address                    | bank number if the section is not |
|      |      |                            | .org, absolute address otherwise  |
+------+------+----------------------------+-----------------------------------+
| 4    | u32  | data size                  |                                   |
+------+------+----------------------------+-----------------------------------+
|      |      | data                       |                                   |
+------+------+----------------------------+-----------------------------------+

Symbol entry:
+======+======+============================+===================================+
| Size | Type | Description                | Value                             |
+======+======+============================+===================================+
| 32   |      | identifier                 | null terminated string            |
+------+------+----------------------------+-----------------------------------+
| 4    | u32  | section id                 | id of the section containing the  |
|      |      |                            | symbol                            |
+------+------+----------------------------+-----------------------------------+
| 2    | u16  | offset                     |                                   |
+------+------+----------------------------+-----------------------------------+
| 1    | u8   | global                     | non-zero means the symbol is      |
|      |      |                            | global                            |
+------+------+----------------------------+-----------------------------------+
| 1    | u8   | extern                     | non-zero means the sybol is       |
|      |      |                            | extern                            |
+------+------+----------------------------+-----------------------------------+
